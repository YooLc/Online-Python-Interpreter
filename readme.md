# 在线 Python 编程 (Online Python Interpreter)
## 功能
- 适用于 Python 教学
- 编辑 Python 代码，并输出结果（支持 Turtle 海龟图）
- 支持截图代码及结果
- 静态页面，可以缓存线下使用
## 说明
- Python 的解释器是依靠 [Skulpt](https://github.com/skulpt/skulpt) 使用 Javascript 模拟的，有可能和 CPython 有区别。
- 目前界面还没写好，不过基础功能是可以使用的
